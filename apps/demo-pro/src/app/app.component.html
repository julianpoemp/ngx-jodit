Angular wrapper for Jodit WYSIWYG editor. It supports Angular >= 12. Fore more information about installation, use, options and events see Github README.

Options

All options from Jodit are supported. The options are set via two-way data-binding. Insert your options as JSON to test it.

For example you can insert {"disabled": true}
